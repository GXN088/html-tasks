# HTML Learning Sandbox

HTMLの基本文法から応用的なレイアウトまで、学習用に作成した小さな課題やコードをまとめたリポジトリです。
一歩ずつステップアップしながら、セマンティックなマークアップの習得を目指しています。

## 📌 このリポジトリについて

フロントエンドエンジニアとしての基礎を固めるため、Web規格に準拠した正しいHTML（文章構造）の書き方を練習しています。
各ディレクトリごとに独立したテーマの課題を配置しています。

## ロードマップと課題一覧

- [x] **01-basic-tags**：見出し、段落、リスト、リンク、画像の挿入
- [x] **02-semantic-html**：`header`, `main`, `footer`, `article`, `section`を使った構造化
- [ ] **03-form-elements**：各種入力フォーム、ボタン、バリデーションの基本
- [ ] **04-tables**：テーブルタグを使ったデータ表示とレスポンシブ対応
- [ ] **05-seo-meta**：metaタグ、OGP設定、ファビコンの設定

*※完了した課題にはチェックを入れて管理しています。*

##  開発環境・動作確認

- **エディタ**: VS Code (Visual Studio Code)
- **推奨拡張機能**: Live Server (ローカルでのリアルタイムプレビュー用)
- **確認ブラウザ**: Google Chrome / Mozilla Firefox

### 実行方法
1. このリポジトリをクローンします。
   ```bash
   git clone https://github.com
   ```
2. VS Codeで対象のフォルダを開きます。
3. 実行したい`index.html`を右クリックし、「**Open with Live Server**」を選択してブラウザで確認します。

## 学んだこと・気づき

*   `<div>`タグを多用せず、意味に合ったセマンティックタグを使う重要性を学びました。
*   画像の`alt`属性はスクリーンリーダーやSEOにとって必須であることを意識しました。
