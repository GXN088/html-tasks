В этом проекте вы создадите страницу регистрации на мероприятие, используя HTML. Страница будет включать заголовок с названием мероприятия, расписание, представленное в виде таблицы, форму регистрации с различными полями ввода и футер со ссылками на социальные сети. Этот проект объединяет все изученные вами концепции для создания полноценной веб-страницы. Создайте пустой HTML-документ с нуля, не забудьте теги html и body.
